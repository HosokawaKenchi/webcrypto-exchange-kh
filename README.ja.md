# webcrypto-exchange-kh

ブラウザ上で安全にファイル暗号化・復号を行うWebアプリケーション

## 概要

- このアプリは静的Webページとして実装されています。

- JavaScript主体のアプリケーションです。
    - 静的Webページとして完結できるため、Node.jsなどのサーバーサイドは不要
    - すべての処理はブラウザ上のローカルメモリで実行される

- ブラウザ上で公開鍵と秘密鍵のペアを生成し、ダウンロードさせる。

- **ハイブリッド暗号化対応**：
    - **AES-256-GCM** でファイルを暗号化（大容量ファイルに対応）
    - **RSA-OAEP** (2048ビット)でAES鍵を暗号化
    - これにより、サイズ制限なく任意のファイルを安全に暗号化可能

- ローカルファイルを指定し、ブラウザ上で公開鍵を用いたハイブリッド暗号化を行い、暗号化ファイルをダウンロード（拡張子は .encrypted）

- .encryptedファイルを指定し、ブラウザ上で秘密鍵を用いた復号を行い、復号ファイルをダウンロード

- 後方互換性：旧形式（RSA-OAEP直接暗号化）のファイルにも対応

- サンプルとして、このレポジトリのindex.htmlはGithub Pagesで公開される

## 暗号化方式

### v2.0（現在）：ハイブリッド暗号化
- ファイル暗号化：AES-256-GCM（256ビット鍵、12バイトランダムIV）
- 鍵暗号化：RSA-OAEP（2048ビット）
- ハッシュ：SHA-256
- 対応ファイルサイズ：無制限

### v1.0（旧形式）：RSA-OAEP直接暗号化
- 復号時のみサポート（後方互換性）
- 対応ファイルサイズ：最大約245バイト

## 使用方法

### 基本フロー
1. **受取人**がこのページでキーペアを生成
2. **受取人**が公開鍵のみを送り手に共有
3. **送り手**が受け取った公開鍵でファイルを暗号化
4. **受取人**が秘密鍵で暗号化ファイルを復号

### PWA（Progressive Web App）機能
このアプリはPWA対応しており、以下の機能が利用可能です：

- **オフラインアクセス**：一度読み込んだ後、インターネット接続がなくても使用可能
- **ホーム画面への追加**：スマートフォンやタブレットにアプリのようにインストール可能
- **ネイティブアプリのような動作**：ブラウザのアドレスバーなしにフルスクリーン表示

#### インストール方法
1. **スマートフォン（iOS）**：
   - Safariで本アプリを開く
   - 共有ボタン（↑）をタップ
   - 「ホーム画面に追加」を選択

2. **スマートフォン（Android）**：
   - Chromeなどで本アプリを開く
   - メニューボタン（⋮）をタップ
   - 「アプリをインストール」または「ホーム画面に追加」を選択

3. **デスクトップ（Windows/Mac）**：
   - Chromeで本アプリを開く
   - アドレスバーの「インストール」ボタンをクリック
