WebCrypto Exchange KH

ブラウザ上での安全なファイル暗号化・復号ツール

1. キーペア生成

公開鍵と秘密鍵のペアを生成します

注意：受け取る側がこのページで鍵ペアを生成し、生成された公開鍵のみを送る側に送付してください。秘密鍵は絶対に他人に共有しないでください。送る側は受け取った公開鍵を使用してファイルを暗号化します。

2. ファイル暗号化（公開鍵使用）

ローカルファイルを選択し、公開鍵で暗号化します

3. ファイル復号（秘密鍵使用）

.wcxファイルを選択し、秘密鍵で復号します

使用方法

公開鍵と秘密鍵の仕組み

公開鍵暗号は、2つの異なるキーを使用する仕組みです。公開鍵は誰にでも公開できるキーで、データの暗号化に使用します。秘密鍵は厳重に保管する必要があるキーで、暗号化されたデータの復号に使用します。

ハイブリッド暗号化の仕組み

このツールはハイブリッド暗号化方式を採用しています：

  1. AES-256-GCM暗号化：大容量ファイルを高速に暗号化するため、ランダムに生成したAES鍵を使用してファイルを暗号化します
  2. RSA-OAEP鍵暗号化：生成したAES鍵を秘密鍵の持ち主のみが復号できるよう、公開鍵でRSA暗号化します

この仕組みにより、フォトグラフ、動画、圧縮ファイルなど、サイズ制限なく安全にファイルを暗号化できます。

このツールでは、受取人が鍵ペアを生成し、公開鍵のみを送り手に送信します。送り手は受け取った公開鍵でファイルを暗号化し、受取人は秘密鍵でそれを復号します。

ステップバイステップ手順

受取人の作業：

  1. このページの「キーペアを生成」ボタンをクリックしてRSA 2048ビットキーペアを生成します
  2. 「公開鍵をダウンロード」ボタンをクリックして、public_key.pubファイルをダウンロードします
  3. 公開鍵ファイルを送り手に安全に送付します
  4. 「秘密鍵をダウンロード」ボタンをクリックして、private_key.pemファイルをダウンロードし、安全に保管します

送り手の作業：

  1. 受け取った公開鍵ファイル（public_key.pub）を「公開鍵ファイルを選択」で選択します
  2. 暗号化したいファイルを「暗号化するファイルを選択」で選択します
  3. 「ファイルを暗号化」ボタンをクリックします
  4. 暗号化されたファイル（ファイル名.wcx）が自動的にダウンロードされます
  5. 暗号化ファイルを受取人に送付します

受取人の復号作業：

  1. 「秘密鍵ファイルを選択」で、保管していたprivate_key.pemを選択します
  2. 「復号するファイル（.wcx）を選択」で、受け取った暗号化ファイルを選択します
  3. 「ファイルを復号」ボタンをクリックします
  4. 復号されたファイルが自動的にダウンロードされます

セキュリティに関する注意

  • すべての処理はブラウザ上で実行されます（サーバーに送信されません）
  • 秘密鍵は絶対に第三者と共有しないでください
  • 秘密鍵は安全な場所に保管してください
  • 複数のコンピュータで使用する場合、秘密鍵を安全に移動させてください
