# webcrypto-exchange-kh

# 概要

- 静的Webページアプリである。

- JavaScript主体のアプリケーションである。
    - 静的Webページとして完結できないNode.jsなどは使わない

- ブラウザ上で公開鍵と秘密鍵のペアを生成し、ダウンロードさせる。

- javascriptのblobを使ってローカルファイルを指定し、ブラウザ上のローカルメモリで公開鍵を用いた暗号化を行い、ダウンロード機能で暗号化したファイルを返す。拡張子は.wcx

- javascriptのblobを使ってローカル.wcxファイルを指定し、ブラウザ上のローカルメモリで秘密鍵を用いた復号を行い、ダウンロード機能で復号したファイルを返す。


- サンプルとして、このレポジトリのindex.htmlはgithub pagesで公開される
